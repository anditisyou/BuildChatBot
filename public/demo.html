Mock User Website

This page simulates a customer-facing website that embeds your chatbot widget. Use it to verify the widget loads, opens, and sends messages to your backend.

How to test

  1. Open this page in your browser: http://localhost:5000/demo.html
  2. Click the chat button in the bottom-right corner
  3. Send a message like hello or order status
  4. Verify the bot responds from /api/chat/:botId

The widget is embedded below using the bot ID from a compiled bot. If the bot ID is invalid, replace it with a real bot ID from your compile result or the dashboard. 

<script src="/widget.js" data-bot-id="bot_3027c18a_mo14g748"></script>
Note: If you compile a new bot and receive a different bot_id, update the script tag above to use the new ID.
